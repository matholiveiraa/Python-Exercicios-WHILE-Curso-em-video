# Python-Exercicios-WHILE-Curso-em-video
Repositório contendo exercícios práticos de Python focados em laços de repetição while, baseados no curso "Python" do Curso em Vídeo. O objetivo é reforçar a lógica de programação, controle de fluxo e manipulação de loops, consolidando a prática passo a passo.

#Conteúdo

Estrutura de repetição while

Condições de parada

Controle de fluxo com break e continue

Exercícios de lógica e desafios progressivos

#Objetivo

Treinar pensamento lógico, resolução de problemas e domínio de loops, preparando para projetos mais complexos e entrevistas de programação.

#Tecnologias

Python 3.x
